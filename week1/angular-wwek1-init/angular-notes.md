# Angular Notes

* Install angular-cli:
  ```bash
  npm install -g @angular/cli@6.2.1
  ```
* Create a new folder and setup angular application
  ```bash
  ng new conFusion --style=scss
  ```
* Move into the `conFusion` folder and run:
  ```bash
  npm install
  ```
* Run the angular cli in-built server
  ```bash
  ng serve
  ```
